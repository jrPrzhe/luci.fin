═══════════════════════════════════════════════════════════════
  СПОСОБ 1: Через веб-интерфейс GitHub (САМЫЙ ПРОСТОЙ)
═══════════════════════════════════════════════════════════════

1. Откройте: https://github.com/jrPrzhe/demo0811
2. Нажмите "Add file" → "Upload files"
3. Перетащите ВСЕ файлы из папки deploy-demo:
   - index.html
   - папку images/ (со всеми 5 PNG файлами)
4. Нажмите "Commit changes"
5. Settings → Pages → Source: main / (root) → Save
6. Готово! Демо будет на: https://jrprzhe.github.io/demo0811/

═══════════════════════════════════════════════════════════════
  СПОСОБ 2: Через Git (для продвинутых)
═══════════════════════════════════════════════════════════════

cd demo0811
# Скопируйте все файлы из deploy-demo в корень репозитория
git add .
git commit -m "Add demo page"
git push origin main
# Затем включите Pages в настройках GitHub
