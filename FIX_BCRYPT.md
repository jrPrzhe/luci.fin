# Исправление проблемы с bcrypt

Проблема связана с версией bcrypt и ограничением длины пароля (72 байта).

## Решение 1: Обновить библиотеки (рекомендуется)

```bash
pip install --upgrade bcrypt passlib[bcrypt]
```

## Решение 2: Использовать исправленную версию

Я уже исправил функцию `get_password_hash` в `app/core/security.py` - она теперь автоматически обрезает пароли длиннее 72 байт.

## Решение 3: Переустановить bcrypt

```bash
pip uninstall bcrypt
pip install bcrypt
```

## Проверка после исправления

После обновления библиотек или применения исправления попробуйте снова:

```bash
python create_test_user.py
```

## Если проблема сохраняется

Попробуйте создать пользователя с более коротким паролем:

```bash
python create_test_user.py --password test123
```

Или используйте API для регистрации пользователя через веб-интерфейс.
